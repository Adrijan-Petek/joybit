# Game Tiles

Place your game tile images here.

## Requirements

- **8 different tile types** (numbered 0-7)
- **Square format** (256x256px recommended)
- **PNG format** with transparency
- **Consistent style** across all tiles

## Naming Convention

```
tile-0.png
tile-1.png
tile-2.png
tile-3.png
tile-4.png
tile-5.png
tile-6.png
tile-7.png (time/bonus tile)
```

## Design Tips

- Use bright, contrasting colors
- Make tiles easily distinguishable
- Consider animations (can be GIFs or sprite sheets)
- Test visibility on different backgrounds

## Example Themes

- **Classic**: Gems and jewels
- **Crypto**: Different cryptocurrency logos
- **Neon**: Glowing geometric shapes
- **Emoji**: Popular emojis
- **Custom**: Your unique design
