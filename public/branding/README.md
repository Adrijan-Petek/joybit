# Branding Assets

Place your branding assets here:

- `logo.png` - Main logo (512x512px recommended)
- `logo-small.png` - Small logo for header (64x64px)
- `favicon.ico` - Favicon
- `banner.png` - Social media banner (1200x630px)
- `app-icon.png` - App icon for mobile (1024x1024px)

## Logo Usage

The logo is used in:
- Header navigation
- Admin panel access trigger (10 clicks)
- Social media sharing
- Mobile app icon

## Design Guidelines

- Use transparent backgrounds for PNG files
- Maintain consistent color scheme (purple/pink gradient)
- Ensure logos are crisp at various sizes
- Include both light and dark variants if needed
